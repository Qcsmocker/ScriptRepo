# Bash Script Usage

## Overview
This project contains a Bash script that can be executed from the terminal.

## Requirements
- Linux / macOS / WSL (Windows Subsystem for Linux)
- Bash installed (usually pre-installed)

## How to Run

### 1. Make the script executable
```bash
chmod +x script.sh
```

### 2. Rune the script
```bash
./script.sh
```
